# Posterior distribution of the number of cells collected in the first and second samples.

Posterior distribution of the number of cells collected in the first and
second samples.

## Usage

``` r
plot_expected_N(x)
```

## Arguments

- x:

  TOSCA object

## Value

Posterior distribution of the number of cells collected in the first and
second samples, considering the growth rate (omega) and the time of
birth of the MRCA (t_mrca) of each sample.
