# Plot clinical timeline + posterior times with MAP

Plot clinical timeline + posterior times with MAP

## Usage

``` r
plot_timing_MAP(x)
```

## Arguments

- x:

  TOSCA obj

## Value

posterior distributions of the inferred timing variables on the clinical
timeline (alternative visualisation)

## Examples

``` r
data("exampleFit")
plot_timing(exampleFit)
```
