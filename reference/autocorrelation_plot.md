# Plot the autocorrelation of MCMC samples with progressive iterations

Plot the autocorrelation of MCMC samples with progressive iterations

## Usage

``` r
autocorrelation_plot(x, parameters = NULL, days_from = NULL)
```

## Arguments

- x:

  TOSCA object

- parameters:

  parameters

- days_from:

  date from which to compute the distance from for times

## Value

plot
