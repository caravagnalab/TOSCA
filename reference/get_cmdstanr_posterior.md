# Get posterior draws from cmdstanr obj

Get posterior draws from cmdstanr obj

## Usage

``` r
get_cmdstanr_posterior(x, days_from = NULL, parameters = NULL)
```

## Arguments

- x:

  TOSCA obj

- days_from:

  date from wich to compute the distance from for times

- parameters:

  parameters one desires to retrive the posterior for

## Value

posterior draws
